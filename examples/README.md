# MeriLang Examples

This directory contains 25+ example programs demonstrating MeriLang features.

## Basic Examples

### Fundamentals (01-06)

- **01_hello_world.meri** - Your first program
- **02_variables.meri** - Variables and data types
- **03_arithmetic.meri** - Math operations
- **04_conditionals.meri** - If-else statements
- **05_while_loop.meri** - While loops
- **06_for_loop.meri** - For each loops

### Functions & Data Structures (07-11)

- **07_functions.meri** - Function definitions
- **08_recursion.meri** - Recursive functions (factorial, fibonacci)
- **09_lists.meri** - List operations
- **10_dictionaries.meri** - Dictionary usage
- **11_lambdas.meri** - Anonymous functions

### Object-Oriented Programming (12-14)

- **12_classes.meri** - Class definitions
- **13_inheritance.meri** - Class inheritance
- **14_error_handling.meri** - Try-catch-finally

## Algorithm Examples

- **20_temperature_converter.meri** - Celsius/Fahrenheit converter

## Running Examples

```bash
# Run any example
python -m MeriLang examples/01_hello_world.meri

# Or using the interpreter directly
python MeriLang/main.py examples/15_fizzbuzz.meri
```
