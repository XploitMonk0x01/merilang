üì¶ Variables

Declaration

maan x = 10
maan name = "Ahmed"
maan isValid = sach  // true

maan declares a variable (Hindi: "value/assumption")

üì¢ Output

Print to Console

likho("Hello, World!")
likho(x)
likho("Sum:", x + y)

likho prints output (Hindi: "write")

üîÄ Conditionals

If-Else

agar x > 10 {
    likho("Greater than 10")
} warna {
    likho("Less than or equal to 10")
}

agar = if, warna = else

Else-If Chain

agar x > 100 {
    likho("Large")
} agarlena x > 50 {
    likho("Medium")
} warna {
    likho("Small")
}

agarlena = else if

üîÅ Loops

While Loop

maan i = 0
jab_tak i < 10 {
    likho(i)
    maan i = i + 1
}

jab_tak = while (literally: "as long as")

For Loop

bar_bar i in [1, 2, 3, 4, 5] {
    likho(i)
}

bar_bar = for (literally: "repeatedly")

Loop Control

bar_bar i in [1, 2, 3, 4, 5] {
    agar i == 3 {
        ruk  // break
    }
    agar i == 2 {
        age_badho  // continue
    }
    likho(i)
}

ruk = break, age_badho = continue

Function Definition

kaam add(a, b) {
    wapas a + b
}

maan result = add(5, 3)
likho(result)  // 8

kaam = function, wapas = return

No Parameters

kaam greet() {
    likho("Namaste!")
}

greet()

üéØ Classes & OOP

Class Definition

class Person {
    kaam __init__(name, age) {
        yeh.name = name
        yeh.age = age
    }
    
    kaam greet() {
        likho("Hello, I'm", yeh.name)
    }
}

class defines a class, yeh = this/self

Creating Objects

maan person = naya Person("Ahmed", 25)
person.greet()
likho(person.name)

naya = new (creates object instance)

Inheritance

class Student badhaao Person {
    kaam __init__(name, age, grade) {
        upar.__init__(name, age)
        yeh.grade = grade
    }
    
    kaam study() {
        likho(yeh.name, "is studying")
    }
}

badhaao = extends, upar = super

üõ°Ô∏è Error Handling

Try-Catch

koshish {
    maan result = 10 / 0
} pakdo e {
    likho("Error:", e)
}

koshish = try, pakdo = catch

Try-Catch-Finally

koshish {
    // risky operation
} pakdo e {
    likho("Error occurred")
} akhir {
    likho("Cleanup code")
}

akhir = finally (always executes)

Throwing Errors

kaam divide(a, b) {
    agar b == 0 {
        fenko "Cannot divide by zero"
    }
    wapas a / b
}

fenko = throw (raises exception)
